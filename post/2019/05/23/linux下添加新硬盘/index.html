查看新硬盘

fdisk -l

对新硬盘进行分区

fdisk /dev/sdb

查看分区

fdisk -l

将分区格式化

mkfs -t ext3 /dev/sdb1 #文件系统根据自己的需要选择

挂载分区

mount /dev/sdb1 /data

添加开机自动挂载

如果要启动时自动挂载，需要编辑/etc/fstab文件

1
2
3
4
5

vim /etc/fstab

#添加以下内容

/dev/sdb1 /data ext3  defaults 0 0

这样以后重启就会自动挂载到/data下
