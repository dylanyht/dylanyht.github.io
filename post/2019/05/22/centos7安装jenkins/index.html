Centos7安装jenkins

2019-05-22
Devops
202 words 1 min read times read

Contents

    • 安装
    • 启动
    • 访问

在安装jenkins之前，将jenkins相关工具集安装完：jdk、maven、git

此文档使用yum方式安装jenkins

安装

  • 下载依赖
1

sudo wget -O /etc/yum.repos.d/jenkins.repo https://pkg.jenkins.io/redhat-stable/jenkins.repo
  • 导入秘钥
1

sudo rpm --import https://pkg.jenkins.io/redhat-stable/jenkins.io.key
  • 安装
1

yum install jenkins

启动

1
2

systemctl start jenkins.service
systemctl enable jenkins.service

访问

web端输入http://:8080

默认端口为8080 可以在/etc/sysconfig/jenkins查看更改

会需要输入密码

1

cat /var/lib/jenkins/secrets/initialAdminPassword  #查看密码

然后根据自己需要进行配置就可以了

如果要深入学习 请参考官方文档
