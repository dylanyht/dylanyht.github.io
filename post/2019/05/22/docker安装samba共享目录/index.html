Docker安装samba共享目录

2019-05-22
容器
181 words 1 min read times read

Contents

    • 查找镜像
    • 拉取使用量最多的镜像
    • 创建共享目录
    • 启动容器

此文档需要在docker安装完后操作 制作一个简易的samba共享

查找镜像

1

docker search samba

拉取使用量最多的镜像

1

docker pull  dperson/samba

创建共享目录

在本地创建目录，以便于容器挂载

1
2

mkdir /home/shared            //在/home目录下创建shared目录
chmod 777 /home/shared  //修改shared权限，不修改的话连接进去会提示没有权限写入数据

启动容器

1
2

docker run -it --name samba -p 139:139 -p 445:445 -v //home/shared:/mount -d dperson/samba -u "bl;bl" -s "shared;/mount/;yes;no;no;all;none"
# 注：-u "user:passwd"  连接时的账号和密码
