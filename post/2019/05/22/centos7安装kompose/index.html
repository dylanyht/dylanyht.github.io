kompose是一个可以在kubernetes运行docker-compose.yml的工具，也可以将docker-compose.yml转换为kubernetes的相应文件，非常便利。

下载安装

下载

1

curl -L https://github.com/kubernetes/kompose/releases/download/v1.18.0/kompose-linux-amd64 -o kompose

安装

1
2

chmod +x kompose sudo
mv ./kompose /usr/local/bin/kompose

使用

1
2

#将docker-compose转成kubernetes yaml
kompose -f docker-compose.yml convert

可能出现的问题

使用时提示错误

1

FATA Version 3.7 of Docker Compose is not supported. Please use version 1, 2 or 3

解决办法

将docker-compose.yml文件的version改成3或3以下
